# inflexa-ai/homebrew-tap

Homebrew formulae for [Inflexa](https://github.com/inflexa-ai/inflexa) tools.

## Install

```bash
brew install inflexa-ai/tap/inflexa
```

Or tap once and install by name:

```bash
brew tap inflexa-ai/tap
brew install inflexa
```

## About the formulae

Formula files in this repository are **generated** by the release automation
in [inflexa-ai/inflexa](https://github.com/inflexa-ai/inflexa) (see
`.github/homebrew/` there) — every release pushes an updated formula here.
Don't edit them by hand; changes belong in the source template.
